# Resumo do sistema
A função do nosso sistema é disponibilizar conteúdos de estudo, como resumos e quiz, para estudantes que estão se preparando para vestibulares como o Provão Paulista. Cada matéria que cai no vestibular possui a sua aba de resumos e quizzes, cada quiz com 5 perguntas. Porém, o diferencial do nosso sistema é que ele também disponibiliza informações sobre quais faculdades aceitam o vestibular, seus cursos mais disputados e quais outros vestibulares também são aceitos para cada uma das universidades.

# Manual de instruções

### Login
1. O usuário deve inserir o nome de usuário e a senha que foram cadastrados.
2. Após confirmar o login, se os dados estiverem corretos, o usuário é direcionado para a tela home.

### Cadastro
1. Caso não tenha uma conta criada, o usuário pode criar uma preenchendo os campos solicitados, que são: nome completo, senha e email de recuperação.
2. Após o cadastro, pode fazer login normalmente.

### Menu Fixo
Depois do cadastro, todas as telas possuem o menu fixo no lado esquerdo da tela.  
O menu redireciona para as outras telas de acordo com o clique, e possui as seguintes telas:  
• Tela de Início  
• Pontuações  
• Assinaturas  
• Provão Paulista (Matérias e Materiais)  
• USP (informações e cursos)  
• UNESP (informações e cursos)  
• UNICAMP (informações e cursos)  
• FATEC (informações e cursos)  

### Tela Inicial
1. Possui uma mensagem de boas vindas para o usuário.
2. No canto superior, tem um botão que abre a tela com informações sobre o vestibular Provão Paulista e as universidades. Cada uma das universidades possui o botão de mais informações, que redireciona o usuário para a tela de informação sobre a faculdade selecionada, e a parte do vestibular possui um botão que redireciona para a tela de conteúdos.

### Universidades
1. Para cada universidade, existem duas telas.
• Tela de Informações: tem dados gerais sobre a instituição, informações sobre vestibulares aceitos e a quantidade de vagas disponibilizadas
• Tela de Cursos: descrição rápida dos cursos que possuem mais demanda na universidade

### Assinaturas
1. Nesta tela, o usuário pode escolher entre:  
Plano Gratuito: acesso gratuito aos materiais por 6 meses, navegação livre.  
Plano Pago: funções do plano gratuito, mas com acesso a todos os materiais de estudo por tempo ilimitado  
2. O usuário pode ver as diferenças entre os planos e selecionar aquele que desejar.

### Matérias
1. Nessa tela, o usuário encontra uma lista de todas as matérias disponíveis.
2. Ao lado de cada matéria, o usuário tem dois botões que pode selecionar o botão resumos ou o botão quizzes.
3. Cada botão leva para sua respectiva tela.

### Escolha de Resumos
1. Nessa tela, o usuário possui 5 opções de resumos de acordo com a matéria que escolheu.
2. Cada matéria tem seus próprios resumos, que possuem textos explicativos para estudo antes de realizar o quiz.
3. Após a finalização do resumo, o usuário é direcionado para uma tela que mostra o total de pontos acumulados pelo resumo concluído e possui dois botões, que redirecionam para a tela de escolha e para a tela de quizzes.

### Escolha de Quizzes
1. Nesta tela, aparecem todos os quizzes relacionados à matéria escolhida.
2. Após selecionar o quiz desejado, o usuário inicia o quiz com 5 perguntas.

### Perguntas do Quiz
1. Cada pergunta é mostrada individualmente.
2. Após confirmar a alternativa selecionada, o usuário avança para a próxima pergunta. 
3. Após a submissão das 5 perguntas, aparece a tela que mostra o total de pontos acumulados por questões respondidas corretamente, e também possui os dois botões que redirecionam para a tela de escolha e para a tela de quizzes.

### Pontuações
1. Essa tela mostra os 5 usuários do sistema com maior pontuação em um ranking.
